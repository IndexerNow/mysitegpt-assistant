## What is MySiteGPT Assistant?

MySiteGPT Assistant trains an AI on your website content and embeds it as a chat widget. Every answer cites the exact page it came from, and when it doesn't know, it says so — never a guess. One script tag to embed. 5-minute setup. 14-day free trial, no card required.

---

## Features

### Answers & Accuracy
- **Cited answers** — every response links [n] markers back to the exact source page
- **Honest fallback** — when the AI doesn't know, it says so instead of hallucinating
- **Multi-source training** — URLs, plain text, Q&A pairs, and PDF uploads
- **Visitor feedback** — thumbs up/down per answer to improve quality

### Embed & Integration
- **One-line embed** — single async `<script>` tag before `</body>`
- **Shadow DOM isolation** — zero CSS conflicts with your existing design
- **No slowdown** — async loading doesn't affect page performance
- Works with: Next.js · React · WordPress · Shopify · Webflow · Framer · Vue · Svelte · Squarespace · Plain HTML

### Languages & Reach
- **50+ languages** — responds in the visitor's language automatically
- Global-ready from day one

### Lead Capture & CRM
- Built-in **lead capture form** inside the chat
- Lead routing and notifications
- **Slack integration** for real-time lead alerts

### API & Webhooks
- **Public REST API** — call your assistant from your own backend
- Hashed API keys for security
- Same cited answers over your own transport
- Webhook support for custom workflows

### Content Sync
- **Automatic content sync** — keeps your assistant up to date as your site changes
- Developer plan: sync every 2 days
- Managed plan: daily sync
